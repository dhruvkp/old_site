BYOP- Bring Your Own Privacy

 Advisor- Dr. Nisheeth Shrivastava, Bell Labs, India (Now at Amazon ML research)

  • BYOP tool is a browser extension (for Firefox and Google Chrome) that allows web users to selectively opt-out of tracking by ad-networks. BYOP shows users topics extracted from their previous history, from which they can mark topics as sensitive Thereafter, the real-time BYOP tool won’t allow the ad networks (e.g. Google) to track the user for any webpage falling under sensitive categories/clusters.
  • Implemented and compared results from Support Vector Machine, Topic Modeling and Naive Bayesian for classification, into categories.
  • Implemented and compared results from KMeans algorithm, K-nearest neighbors using Kd-tree, Carrot clustering tool and a custom algorithm based on term vectors using unigrams and bigrams for clustering of documents.
  • Project Thesis
